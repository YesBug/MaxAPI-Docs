title: "GoAMZ"
description: "Configure MaxAPI for GoAMZ chat, Midjourney, Kling, Luma, and Suno modules."
---

Use the following MaxAPI addresses in the matching GoAMZ modules:

| Module | Address |
| --- | --- |
| Chat / OneAPI | `https://api.maxapi.ai` |
| Midjourney | `https://api.maxapi.ai/mj` |
| Midjourney Fast | `https://api.maxapi.ai/mj-fast/mj` |
| Midjourney Relax | `https://api.maxapi.ai/mj-relax/mj` |
| Kling | `https://api.maxapi.ai/kling/v1` |
| Luma | `https://api.maxapi.ai` |
| Suno | `https://api.maxapi.ai/suno/v1` |

<Note>Restart the relevant account pool after changing an endpoint.</Note>
