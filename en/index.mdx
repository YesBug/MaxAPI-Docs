## Base URL

Send API requests to:

```text
https://api.maxapi.ai
```

Most text models are compatible with the OpenAI API. Change the base URL, API key, and model ID to migrate an existing application.

<Tip>
  Model availability changes over time. Use the model list in the MaxAPI console as the source of truth.
</Tip>

## Choose an API format

<AccordionGroup>
  <Accordion title="OpenAI-compatible format">
    Use this format for chat, completions, embeddings, images, and audio. Existing OpenAI applications usually require only configuration changes.
  </Accordion>
  <Accordion title="Unified asynchronous format">
    Use this format for operations that take longer to complete. Create a task, then poll its status with the returned task ID.
  </Accordion>
  <Accordion title="Provider-native format">
    Use a native format when you need advanced provider-specific parameters or want to migrate an existing provider integration.
  </Accordion>
</AccordionGroup>
