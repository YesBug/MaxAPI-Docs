## Base URL

所有请求均发送至：

```text
https://api.maxapi.ai
```

大多数文本模型兼容 OpenAI API。你只需替换 `base_url`、API Key 和模型名称。

<Tip>
  模型名称与可用能力会持续更新。请以 MaxAPI 控制台中的模型列表为准。
</Tip>

## 选择接口格式

<AccordionGroup>
  <Accordion title="OpenAI 兼容格式">
    适合聊天、补全、嵌入、图像和音频等常见场景。已有 OpenAI 项目通常只需修改 Base URL 和模型名称。
  </Accordion>
  <Accordion title="统一异步格式">
    适合生成时间较长的图像、视频或音乐任务。创建任务后，使用任务 ID 轮询结果。
  </Accordion>
  <Accordion title="厂商原生格式">
    适合需要特定模型高级参数或迁移已有厂商 SDK 的场景。
  </Accordion>
</AccordionGroup>
